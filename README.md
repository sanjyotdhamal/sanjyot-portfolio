## 🚀 About This Project

This is my **personal portfolio website** — built to showcase my projects, skills, certifications, experience, and athletic achievements. Designed with a clean, minimal aesthetic and packed with modern animations and interactions.

🌐 **Live Site:** [sanjyot-portfolio.netlify.app](https://sanjyot-portfolio.netlify.app)

---

## ✨ Features

- 🎨 **Animated Hero** — Typing animation cycling through roles + floating particles background
- 🖱️ **Interactive Background** — Aurora blobs, mouse glow, geometric shapes & click ripples
- 📦 **3D Project Cards** — Tilt effect on hover with alternating scroll animations
- 🏆 **Certification Marquee** — Dual-row infinite scroll with hover color effects
- 📊 **Animated Skills** — Scrolling marquee tags inside each category
- ⏱️ **Timeline Sections** — Animated timeline for Education & Experience
- 📱 **Responsive Design** — Works on all screen sizes
- ⚡ **Smooth Animations** — Staggered fade-ins, slide-ins throughout all sections

## 🏃 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/sanjyotdhamal/sanjyot-portfolio.git

# 2. Navigate to project directory
cd sanjyot-portfolio

# 3. Install dependencies
npm install

# 4. Start development server
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) in your browser.

### Build for Production

```bash
npm run build
```
